Classification Model Methodology

Data Preprocessing

  • 6,497 wine samples analyzed
  • Missing value imputation using mean strategy
  • Correlation analysis and feature selection
  • MinMax normalization for optimal performance
  • Binary classification: Quality > 5 = Good

Model Training

  • Multiple algorithms tested and compared
  • XGBoost classifier achieved best performance
  • 80/20 train-test split with random state 40
  • ROC-AUC evaluation metric for performance
  • Cross-validation for model robustness

Feature Engineering

  • Wine type encoding (white=1, red=0)
  • Quality threshold optimization
  • Feature importance analysis performed
